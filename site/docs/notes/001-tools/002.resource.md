####  Ikonate

定制、调整和下载免费的矢量图标。Ikonate 是一套适应性强的优化SVG图标，可以在Sketch和Photoshop等开发和设计应用程序中轻松使用。可作为图像、内联SVG或SVG精灵使用。

链接：https://ikonate.com/

####  Feather Icons

Feather 是一个简单漂亮的开源图标集合。每个图标都是在24×24的网格上设计的，强调简单性、一致性和可读性。

链接：https://feathericons.com/
